# Puzzle Event Editor

このツールは、パズルの開始時（Intro）やクリア後（Outro）に挿入される「パズル専用イベント」を作成・編集するためのエディタです。

## ツールの起動方法

`run_puzzle_event_editor.bat` をダブルクリックして起動します。

## 使い方

エディタは大きく3つのペインに分かれています。

1. **レベル一覧 (左ペイン)**
   - 全レベルのリストが表示されます。
   - `[Level XX: Title]` を選択すると、そのレベルに設定されている Intro イベントと Outro イベントのイベントIDを確認できます。
   - レベルを選択した状態で下部の「+ Intro作成」「+ Outro作成」ボタンを押すと、自動的にそのレベル用のイベントID（例：`evt_pz0_intro`）が生成され、編集を開始できます。

2. **イベント構成 (中央ペイン)**
   - 選択したイベントのスクリプト（行単位のコマンド）が表示されます。
   - ここではイベントのセリフ、キャラクターの表情変更、背景変更、ウェイトなどのコマンドを順番に並べます。
   - 「行追加」「行削除」「上へ」「下へ」ボタンを使って、イベントの流れを組み立てます。

3. **行編集 (右ペイン)**
   - 中央ペインで選択した行の詳細（誰が喋るのか、どの立ち絵を表示するのか、テキストの内容など）を編集します。
   - `MSG` 行を選択した場合、キャラクターID、表情、名前、テキストを入力できます。

## 仕組み

- このエディタで保存したデータは、`Asset/fixdata/puzzle_events/` フォルダ内に保存されます。
- ゲームをビルドする際、自動的に `generate_fix_data.py` が走り、これらのイベントデータがゲーム本編（ROM）に組み込まれます。
- パズル本編では、対象レベルが開始されたときや、`CLEAR!`が表示された直後に、対応するイベントが**一度だけ**自動的に再生されます。
